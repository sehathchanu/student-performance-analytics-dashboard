# student-performance-analytics-dashboard

## Overview

This project is an interactive Power BI dashboard designed to analyze student academic performance. The dashboard provides insights into student scores, demographic factors, parental education levels, and test preparation effects.

## Features

- Student Count KPI
- Average Reading, Writing,Math Score
- Overall Average Score Percentage
- Performance Analysis by Ethnicity
- Performance Analysis by Gender
- Parental Education Impact Analysis
- Test Preparation Insights
- Top 5 Students by Performance
- Interactive Filters (Slicers)

## Tools Used

- Microsoft Power BI
- DAX
- Data Modeling
- Data Visualization Techniques

## Key Insights

- Compare performance across different ethnic groups.
- Analyze score variations between male and female students.
- Understand the relationship between parental education and student achievement.
- Evaluate the impact of test preparation on performance.
- Identify top-performing students.
